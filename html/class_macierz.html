Opis szczegółowy

template<unsigned int rozmiar_macierzy, class typ_wartosci>
class Macierz< rozmiar_macierzy, typ_wartosci >

Klasa modelująca macierz kwadratową o zadanym rozmiarze i zadanym typie przechowywanych wartości.

Parametrami szablonu są <rozmiar macierzy, typ wartości>, gdzie: -rozmiar macierzy oznacza ilość wierszy i kolumn wartości przechowywanych przez daną macierz (np. dla macierzy 2x2 wartość 2), -typ wartości oznacza typ zmiennych, które mają być przechowywane przez daną macierz (np. double dla liczb zmiennoprzecinkowych).

Kody wyjątków rzucanych przez klasę: 200 - jako argument operatora funkcyjnego podano wartość wykraczającą poza zakres [0, rozmiar macierzy - 1], 201 - na przekątnej macierzy znajduje się wartość 0; nie można obliczyć wartości wyznacznika macierzy,

Kontener przechowujący opis łączenia wierzchołków w celu tworzenia poszczególnych ścian.

Kontener przechowuje tablice cztero-elementowe przechowujące numery wierzchołków tworzących daną ścianę. Numery wierzchołków należy podawać w następującej kolejności:

[2] - - - [3] | | | | [0] - - - [1]

Opis szczegółowy

template<unsigned int rozmiar_wektora, class typ_wartosci>
class Wektor< rozmiar_wektora, typ_wartosci >

Klasa modelująca wektor o zadanym rozmiarze i zadanym typie przechowywanych wartości.

Parametrami szablonu są <rozmiar wektora, typ wartości>, gdzie: -rozmiar wektora oznacza ilość wartości przechowywanych przez dany wektor (np. dla wektora w przestrzenia 2D wartość 2), -typ wartości oznacza typ zmiennych, które mają być przechowywane przez dany wektor (np. double dla liczb zmiennoprzecinkowych).

Kody wyjątków rzucanych przez klasę: 100 - jako argument operatora indeksującego podano wartość wykraczającą poza zakres [0, rozmiar wektora - 1],
